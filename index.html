The Design Process

Every version follows the same cycle — but the output is wildly different each time.

01

Pick a Direction

Choose a radically different aesthetic — brutalist, retro, neon, editorial, playful. Constraint breeds creativity.

02

Design the System

Define the typography, color palette, spacing, and layout rules. Every design choice should reinforce the mood.

03

Build It

HTML, CSS, and JS — from scratch each time. No templates, no shortcuts. Every pixel is intentional.

04

Ship & Reflect

Publish, compare against previous versions, document what worked and what didn't. Then do it again.
